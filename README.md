# 🎨 Pixel Playground

A mini MS Paint-like browser app built using HTML5 Canvas, CSS, and JavaScript.

## 🚀 Features

- Draw using mouse or touch
- Choose your favorite color
- Clear the canvas anytime
- Responsive and minimal UI

## 🔧 Tech Stack

- HTML5
- CSS3
- JavaScript (Canvas API)
